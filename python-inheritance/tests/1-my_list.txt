The ``MyList`` module
======================

Using ``MyList``
----------------

Import the class:

    >>> MyList = __import__('1-my_list').MyList

Test 1: Basic functionality
    >>> my_list = MyList()
    >>> my_list.append(1)
    >>> my_list.append(4)
    >>> my_list.append(2)
    >>> my_list.append(3)
    >>> my_list.append(5)
    >>> print(my_list)
    [1, 4, 2, 3, 5]
    >>> my_list.print_sorted()
    [1, 2, 3, 4, 5]
    >>> print(my_list)
    [1, 4, 2, 3, 5]

Test 2: Empty list
    >>> my_list = MyList()
    >>> my_list.print_sorted()
    []

Test 3: Single element
    >>> my_list = MyList()
    >>> my_list.append(1)
    >>> my_list.print_sorted()
    [1]

Test 4: Negative numbers
    >>> my_list = MyList()
    >>> my_list.append(-5)
    >>> my_list.append(3)
    >>> my_list.append(-1)
    >>> my_list.append(0)
    >>> my_list.print_sorted()
    [-5, -1, 0, 3]

Test 5: All same numbers
    >>> my_list = MyList()
    >>> my_list.append(5)

    >>> my_list.append(5)
    >>> my_list.print_sorted()

Test 6: Already sorted list
